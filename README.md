# docxside-pdf

Rust library for converting DOCX files to PDF.

## Goal

Produce PDF output that matches Microsoft Word's PDF export as closely as possible.

## Status

Early development.

## Approach

- Parse DOCX (Office Open XML) format
- Render to PDF following Word's layout and formatting rules
- Test against corpus of real-world DOCX files with Word-generated reference PDFs
- Visual comparison testing to verify pixel-level accuracy
